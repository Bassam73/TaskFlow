# TaskFlow - Modern Productivity Landing Page

TaskFlow is a high-performance, fully responsive landing page designed for modern SaaS productivity tools. Built with a focus on "Mobile-First" approach.

[TaskFlow Preview](https://task-flow-steel-five.vercel.app/)
## Key Features

- **Mobile-First Architecture:** Designed for mobile performance and scaled up for desktop users.
- **Responsive Navigation:** Interactive hamburger menu with seamless transitions for mobile devices.
- **Modern Design System:** Utilizes a custom Indigo-based color palette and the Inter typeface for maximum readability.
- **Performance Optimized:** Pure Vanilla CSS and JS with zero external library dependencies for lightning-fast load times.
- **Accessible UI:** Semantic HTML5 structure for better SEO and screen-reader compatibility.

## 🛠 Tech Stack

- **HTML5:** Semantic markup structure.
- **CSS3:** Advanced Flexbox layouts, CSS Variables (Custom Properties), and Positioning (Sticky/Absolute/Fixed).
- **Vanilla JavaScript:** Light-weight DOM manipulation for interactive UI components.
